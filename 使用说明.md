# 🚀 NovaProxy 使用说明

**NovaProxy** 是一款专为开发者设计的极简代理工具，它可以将 Claude (Anthropic) 的 API 请求转换并转发至 NVIDIA 的高性能模型接口。支持可视化配置、自动重连以及多密钥管理。

## ✨ 核心特性

- **极致简约**：全中文管理界面，支持启动后自动弹出浏览器后台。
- **高级配色**：暖沙色调设计，视觉温润、舒适。
- **动态配置**：无需重启，在界面修改 API Key 立即生效，并自动保存到本地。
- **安全隔离**：配置与程序分离，支持设置私有访问暗号，防止接口被盗用。

## 🛠️ 安装与启动

1. **下载/定位文件**：找到 `NovaProxy.exe`。
2. **启动程序**：双击运行 `NovaProxy.exe`。
3. **后台配置**：
   - 启动后，浏览器会自动打开 `http://localhost:3001`。
   - 在 **“设置中心”** 填入您的 **NVIDIA API Key**（在 nvidia build 生成）。
   - 点击 **“保存配置”**。

## 🔌 客户端接入方法

### 1. Claude Code (命令行)
在启动 `claude` 之前，请在终端执行以下命令：

```powershell
# Windows PowerShell
$env:ANTHROPIC_BASE_URL="http://localhost:3001/v1"
$env:ANTHROPIC_API_KEY="任意字符" # 如果在 NovaProxy 界面设置了访问验证，请填写该暗号
claude
```

### 2. Cursor / Chatbox / 其他 UI 工具
- **模型供应商**：选择 `Claude`（路径通常会自动补全）
- **API 地址 (Base URL)**：`http://localhost:3001/v1`
- **API Key**：填入您在后台设置的访问验证密钥（未设置则随便填）。

## 📁 文件说明
- `NovaProxy.exe`：主程序，可直接考贝到任何文件夹使用。
- `config.json`：您的私有配置文件，由程序在第一次保存设置时自动创建。

## ❓ 常见问题
- **地址无效**：请优先使用 `http://localhost:3001` 而非 `127.0.0.1`，以避免本地 VPN 的代理拦截。
- **端口冲突**：如无法启动，请检查 3001 端口是否被占。
